PROJECT OVERVIEW
================================================================================

MONGODB SCHEMAS TO CREATE
================================================================================

Create Mongoose schemas with TypeScript interfaces for all of the following:

---- Organization (tenant root) ----
  name, slug, plan ('starter'|'professional'|'enterprise'),
  modules: string[], billingEmail, stripeCustomerId,
  employeeCount, industry, createdAt

---- User ----
  organizationId (ref), email, passwordHash,
  role ('admin'|'hr_manager'|'manager'|'coachee'|'coach'),
  firstName, lastName, isActive, lastLoginAt

---- SurveyTemplate ----
  organizationId, moduleType ('conflict'|'neuroinclusion'|'succession'),
  title, questions: [{id, text, type ('scale'|'text'|'boolean'), category}],
  isActive, createdBy

---- SurveyResponse ----
  organizationId, templateId, respondentId, departmentId,
  responses: [{questionId, value}], submittedAt, isAnonymous

---- ConflictAnalysis ----
  organizationId, surveyPeriod, departmentId,
  riskScore (0-100), riskLevel ('low'|'medium'|'high'|'critical'),
  conflictTypes: string[], aiNarrative, managerScript,
  escalationRequested: boolean, escalationStatus, createdAt

---- DevelopmentPlan (IDP) ----
  organizationId, coacheeId, coachId,
  goal, currentReality, options: string[], willDoActions: string[],
  milestones: [{title, dueDate, status, notes}],
  eqiScores: object, competencyGaps: string[],
  aiGeneratedContent, status ('draft'|'active'|'completed'), createdAt

---- NeuroinclusiontAssessment ----
  organizationId, respondentRole,
  dimensions: [{name, score, responses: object}],
  overallMaturityScore, aiGapAnalysis, actionRoadmap: string[],
  completedAt

KEY CONSTRAINTS — READ CAREFULLY
================================================================================

MULTI-TENANCY IS CRITICAL
  Every database query MUST include an organizationId filter.
  Add a Mongoose middleware/plugin that enforces this automatically.

NO INDIVIDUAL-LEVEL REPORTING
  Survey response aggregation must enforce a minimum group size of 5
  before returning results to managers.

TYPESCRIPT STRICT MODE
  Enable "strict": true in both backend and frontend tsconfig files.

ERROR HANDLING
  - Global error handler middleware in Express
  - HTTP interceptor in Angular that handles 401 and the refresh token flow

README.md
  Write a complete developer setup guide including:
  - MongoDB Atlas cluster setup steps
  - Environment variable configuration
  - Local dev startup commands (docker-compose and manual)

AWS DEPLOYMENT
  Target EC2 for initial deployment.
  Structure the code so migration to ECS/Fargate requires minimal changes.

ARCHITECTURAL DECISIONS
