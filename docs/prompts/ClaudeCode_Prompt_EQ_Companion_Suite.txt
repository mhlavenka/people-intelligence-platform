================================================================================

Helena Hlavenkova is a PCC-certified executive coach and certified EQi 2.0
administrator. She currently pays MHS (Multi-Health Systems) $275–$700 CAD per
person per assessment. This module eliminates that cost in two phases:

  Phase 1 — EQi Companion App (build now)
    Works ALONGSIDE MHS assessments. Helena imports scores from MHS PDF reports

    branded PDF reports. Zero dependency on MHS portal for coaching workflow.

  Phase 2 — HEQ™ Native Assessment (scaffold now, build later)
    Helena's own proprietary 72-item EI assessment. Replaces MHS entirely.

trademarks (EQi, EQ-i 2.0, EQ 360), or their scoring algorithms. We use the
publicly published Bar-On theoretical framework (5 composites / 15 subscales)
which is documented in peer-reviewed academic literature and is not proprietary.

---- EqProfile ----
interface EqProfile {
  _id: ObjectId
  organizationId: ObjectId        // required — tenant isolation
  coachId: ObjectId               // Helena or delegated coach
  clientName: string
  clientEmail: string
  clientRole: string              // e.g. "VP Operations"
  industry: string
  coachingGoals: string[]
  assessmentHistory: ObjectId[]   // refs to EqScores
  sessionHistory: ObjectId[]      // refs to EqSessions
  createdAt: Date
  updatedAt: Date
}

---- EqScores ----
interface EqScores {
  _id: ObjectId
  profileId: ObjectId
  organizationId: ObjectId
  assessmentType: 'mhs_imported' | 'heq_native'
  assessmentDate: Date
  source: string                  // e.g. "MHS Workplace Report"
  // 5 composite scores (standard scores, mean=100 SD=15)
  composites: {
    selfPerception: number
    selfExpression: number
    interpersonal: number
    decisionMaking: number
    stressManagement: number
  }
  // 15 subscale scores
  subscales: {
    // Self-Perception
    selfRegard: number
    selfActualization: number
    emotionalSelfAwareness: number
    // Self-Expression
    emotionalExpression: number
    assertiveness: number
    independence: number
    // Interpersonal
    interpersonalRelationships: number
    empathy: number
    socialResponsibility: number
    // Decision Making
    problemSolving: number
    realityTesting: number
    impulseControl: number
    // Stress Management
    flexibility: number
    stressTolerance: number
    optimism: number
  }
  totalEI: number
  wellBeingIndicator: number      // optional — MHS provides this
  validityFlags: {
    positiveImpression?: number
    negativeImpression?: number
    inconsistencyIndex?: number
  }
  normGroup: string               // e.g. "General Population", "Professional Global"
  reportNotes: string             // Helena's private notes on this assessment
}

---- EqSession ----
interface EqSession {
  _id: ObjectId
  profileId: ObjectId
  organizationId: ObjectId
  sessionDate: Date
  sessionNumber: number
  sessionTheme: string
  subscalesFocus: string[]        // which subscales were discussed
  coachNotes: string              // Helena's private notes
  clientInsights: string          // key insights from client
  milestoneUpdates: [{
    description: string
    status: 'not_started' | 'in_progress' | 'completed'
    dueDate: Date
  }]
  nextSessionFocus: string
  aiPromptsUsed: string[]         // which AI prompts were generated this session
}

---- EqReport ----
interface EqReport {
  _id: ObjectId
  profileId: ObjectId
  scoreId: ObjectId
  organizationId: ObjectId
  reportType: 'individual_debrief' | 'grow_plan' | 'team_profile' | 'progress_review'
  generatedAt: Date
  pdfS3Key: string                // S3 object key for the PDF
  pdfUrl: string                  // pre-signed URL (refreshed on access)
  aiNarrative: {
    overallSummary: string
    topStrengths: string[]        // 3 strengths with narrative
    developmentAreas: string[]    // 3 development areas with narrative
    interconnectionAnalysis: string
    leadershipImplications: string
    wellBeingNarrative: string
    coachingFocusSuggestion: string
  }
  growPlan: {
    goals: [{
      subscale: string
      goalStatement: string
      currentReality: string
      options: string[]
      willDoAction: string
      targetDate: Date
      status: 'active' | 'completed'
    }]
  }
  coachEdits: string              // Helena's manual edits to AI output
}

---- HeqResponse (Phase 2) ----
interface HeqResponse {
  _id: ObjectId
  profileId: ObjectId
  organizationId: ObjectId
  completedAt: Date
  responses: [{
    itemId: string
    value: number                 // 1–6 Likert
  }]
  rawScores: Record<string, number>    // keyed by subscale name
  tScores: Record<string, number>      // normalized T-scores
  normGroup: string
  completionTimeSeconds: number
}

---- HeqItem (Phase 2) ----
interface HeqItem {
  _id: ObjectId
  itemId: string                  // e.g. "SP-SR-01"
  composite: string               // e.g. "selfPerception"
  subscale: string                // e.g. "selfRegard"
  text: string                    // the question text (Helena writes these)
  reverseScored: boolean
  version: string                 // "1.0", "1.1" etc.
  validatedAt: Date
  active: boolean
}

---- HeqNorms (Phase 2) ----
interface HeqNorms {
  normGroup: string
  n: number                       // sample size
  means: Record<string, number>   // mean per subscale
  sds: Record<string, number>     // SD per subscale
  updatedAt: Date
}

Use this exact text as the system prompt for every EQ module API call:

"You are an expert emotional intelligence coach and organizational psychologist
working within the Helena Coaching methodology. You specialize in interpreting
and coaching around the Bar-On model of emotional intelligence (5 composites,
15 subscales). Your role is to generate coaching-oriented interpretations —
not clinical diagnoses — that are warm, empathetic, strengths-based, and

Key methodology principles to always apply:

- Ground suggestions in real workplace and leadership scenarios

  115+   : Significant strength — leverage and mentor others
  105–114: Strength — well-developed, actively used

  85–94  : Development opportunity — with awareness, coachable
  Below 85: Priority development area — may significantly impact effectiveness

Always output in clean, structured prose. Use headers where specified."


---- PROMPT TEMPLATES — implement as template literal functions ----

buildDebriefPrompt(scores, profile):
  Include all 15 subscale scores and 5 composite scores in the prompt.
  Request: overall summary, top 3 strengths with narrative, top 3 development
  areas with narrative, interconnection analysis (how scores relate to each
  other), leadership/workplace implications based on client role, and a
  suggested coaching focus for the engagement.
  Format the scores as a readable list, not raw JSON.

buildGrowPlanPrompt(scores, goals, profile):
  Include the client's stated coaching goals alongside their 3 lowest subscale
  scores. Request a GROW plan with 3 development goals, each containing:
  a goal statement, current reality grounded in their actual score, 5 concrete
  development options, and a suggested will-do commitment statement.

buildCoachingQuestionsPrompt(subscales, context):
  Specify the subscale(s) by name and definition. Include any context about
  the client's role, challenges, or session theme. Request 8 powerful coaching
  questions that would surface insight specifically for someone with this
  particular EI profile pattern.

buildTeamNarrativePrompt(teamScoresArray, teamContext):
  Provide aggregated mean scores per subscale for the team. Request a team
  narrative covering: collective EI strengths, shared development areas,
  team dynamics implications, and 3 facilitation/intervention recommendations.
  CRITICAL: Never identify individual team members in the output.

buildProgressReviewPrompt(sessions, baseline):
  Provide session themes, notes summary, and baseline scores. Request a
  progress narrative covering: what has shifted, what patterns are emerging,
  and 2–3 suggested focus areas for upcoming sessions.
